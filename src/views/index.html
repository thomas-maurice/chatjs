What is chatjs ?

Chatjs is a very basic one chatroom web application. It means that all the users connecting it are actually accessing the same chatroom at the same time. It might seem strange though, so one of the major improvements to come is the support for several chatrooms, stay tuned ;)

Chatjs provides some cool features, such as :

  • User count and list
  • Nickname change
  • Color change
  • Ninja mode
  • Status updates
  • Markdown formating
  • Smileys
But you might want to read the provided help in the chat window.

Warning: This software is still very young. It might be slightly unstable so you might experience some bugs frome time to time. Don't panic they will be fixed soon, it would be really helpful if you could fill issues on Github to help solve the problems.
